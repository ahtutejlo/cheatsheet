ua_answer: |
  **Блокчейн** -- це розподілена база даних (реєстр), яка зберігає інформацію у вигляді ланцюжка блоків. Кожний блок містить набір транзакцій, хеш попереднього блоку та власний хеш, що робить ланцюжок незмінним.

  Ключові властивості блокчейну:
  - **Децентралізація** -- дані зберігаються на багатьох вузлах мережі, а не на одному сервері
  - **Незмінність** -- після додавання блоку його неможливо змінити без зміни всіх наступних блоків
  - **Прозорість** -- всі транзакції доступні для перевірки учасниками мережі
  - **Консенсус** -- вузли мережі повинні погодитися щодо стану реєстру

  Блокчейн використовується для криптовалют (Bitcoin, Ethereum), смарт-контрактів, управління ланцюгами постачання, цифрових ідентичностей та багатьох інших застосувань.

  Кожний блок типово містить: **заголовок** (хеш попереднього блоку, timestamp, nonce) та **тіло** (список транзакцій, корінь дерева Меркла).
en_answer: |
  **Blockchain** is a distributed database (ledger) that stores information as a chain of blocks. Each block contains a set of transactions, the hash of the previous block, and its own hash, making the chain immutable.

  Key properties of blockchain:
  - **Decentralization** -- data is stored across many network nodes, not on a single server
  - **Immutability** -- once a block is added, it cannot be changed without altering all subsequent blocks
  - **Transparency** -- all transactions are available for verification by network participants
  - **Consensus** -- network nodes must agree on the state of the ledger

  Blockchain is used for cryptocurrencies (Bitcoin, Ethereum), smart contracts, supply chain management, digital identities, and many other applications.

  Each block typically contains: a **header** (previous block hash, timestamp, nonce) and a **body** (list of transactions, Merkle tree root).
