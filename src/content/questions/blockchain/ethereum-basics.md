en_question: "What is Ethereum and how does it differ from Bitcoin?"
ua_answer: |
  **Ethereum** -- це децентралізована платформа з відкритим кодом, яка дозволяє створювати та виконувати смарт-контракти та децентралізовані застосунки (dApps). Створена Віталіком Бутеріним у 2015 році.

  Основні відмінності від Bitcoin:
  - **Призначення**: Bitcoin -- цифрова валюта та сховище вартості; Ethereum -- платформа для програмованих контрактів
  - **Мова програмування**: Bitcoin має обмежену скриптову мову; Ethereum використовує **Solidity** -- Тюрінг-повну мову
  - **Консенсус**: Bitcoin використовує PoW; Ethereum перейшов на **PoS** (The Merge, 2022)
  - **Час блоку**: Bitcoin -- ~10 хвилин; Ethereum -- ~12 секунд
  - **Віртуальна машина**: Ethereum має **EVM** (Ethereum Virtual Machine), яка виконує код смарт-контрактів

  Ethereum ввів концепцію **Gas** -- одиниці обчислювальної роботи, яка визначає вартість виконання операцій. Це запобігає нескінченним циклам та спаму у мережі.

  Нативна криптовалюта Ethereum -- **Ether (ETH)**, яка використовується для оплати комісій за транзакції та виконання смарт-контрактів.
en_answer: |
  **Ethereum** is a decentralized open-source platform that enables the creation and execution of smart contracts and decentralized applications (dApps). It was created by Vitalik Buterin in 2015.

  Key differences from Bitcoin:
  - **Purpose**: Bitcoin -- digital currency and store of value; Ethereum -- platform for programmable contracts
  - **Programming language**: Bitcoin has a limited scripting language; Ethereum uses **Solidity** -- a Turing-complete language
  - **Consensus**: Bitcoin uses PoW; Ethereum transitioned to **PoS** (The Merge, 2022)
  - **Block time**: Bitcoin -- ~10 minutes; Ethereum -- ~12 seconds
  - **Virtual machine**: Ethereum has the **EVM** (Ethereum Virtual Machine) that executes smart contract code

  Ethereum introduced the concept of **Gas** -- a unit of computational work that determines the cost of executing operations. This prevents infinite loops and network spam.

  Ethereum's native cryptocurrency is **Ether (ETH)**, used to pay transaction fees and execute smart contracts.
