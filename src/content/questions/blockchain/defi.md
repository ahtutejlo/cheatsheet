ua_answer: |
  **DeFi** (Decentralized Finance) -- це екосистема фінансових застосунків, побудованих на блокчейні, які працюють без традиційних посередників (банків, бірж, страхових компаній). Більшість DeFi-протоколів працюють на Ethereum.

  Основні категорії DeFi:
  - **DEX** (децентралізовані біржі) -- обмін токенів без посередників (Uniswap, SushiSwap)
  - **Lending/Borrowing** -- кредитування та позики під заставу криптоактивів (Aave, Compound)
  - **Stablecoins** -- токени з прив'язкою до фіатних валют (DAI, USDC)
  - **Yield Farming** -- заробіток через надання ліквідності протоколам
  - **Insurance** -- децентралізоване страхування (Nexus Mutual)

  **Переваги DeFi**: доступність без KYC, прозорість коду, програмованість, глобальний доступ 24/7.

  **Ризики DeFi**: вразливості смарт-контрактів, impermanent loss при наданні ліквідності, волатильність застав, регуляторна невизначеність.

  Загальна заблокована вартість (**TVL** -- Total Value Locked) -- це ключова метрика, що відображає обсяг коштів у DeFi-протоколах.
en_answer: |
  **DeFi** (Decentralized Finance) is an ecosystem of financial applications built on blockchain that operate without traditional intermediaries (banks, exchanges, insurance companies). Most DeFi protocols run on Ethereum.

  Main DeFi categories:
  - **DEX** (decentralized exchanges) -- token swaps without intermediaries (Uniswap, SushiSwap)
  - **Lending/Borrowing** -- lending and borrowing against crypto collateral (Aave, Compound)
  - **Stablecoins** -- tokens pegged to fiat currencies (DAI, USDC)
  - **Yield Farming** -- earning returns by providing liquidity to protocols
  - **Insurance** -- decentralized insurance (Nexus Mutual)

  **DeFi advantages**: accessibility without KYC, code transparency, programmability, global 24/7 access.

  **DeFi risks**: smart contract vulnerabilities, impermanent loss when providing liquidity, collateral volatility, regulatory uncertainty.

  Total Value Locked (**TVL**) is the key metric reflecting the amount of funds in DeFi protocols.
