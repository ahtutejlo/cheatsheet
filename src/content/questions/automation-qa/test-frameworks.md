ua_answer: |
  Фреймворки для тестування надають структуру, інструменти та бібліотеки для написання та запуску автоматизованих тестів.

  **Java фреймворки:**
  - **JUnit 5** -- стандартний фреймворк для юніт-тестування в Java
  - **TestNG** -- розширений фреймворк з підтримкою паралельного запуску та data providers
  - **Selenium** -- автоматизація веб-браузерів
  - **REST Assured** -- тестування REST API

  **JavaScript/TypeScript фреймворки:**
  - **Playwright** -- сучасний інструмент для E2E тестування
  - **Cypress** -- E2E тестування з вбудованим дебагером
  - **Jest** -- юніт-тестування JavaScript
  - **Vitest** -- швидкий фреймворк для Vite проектів

  **Python фреймворки:**
  - **pytest** -- найпопулярніший Python тестовий фреймворк
  - **Robot Framework** -- keyword-driven тестування

  **Мобільне тестування:**
  - **Appium** -- кросплатформна автоматизація мобільних додатків
  - **Espresso** -- нативне тестування Android
  - **XCUITest** -- нативне тестування iOS

  Вибір фреймворку залежить від мови програмування проекту, типу тестування та потреб команди.
en_answer: |
  Testing frameworks provide structure, tools, and libraries for writing and running automated tests.

  **Java frameworks:**
  - **JUnit 5** -- standard unit testing framework for Java
  - **TestNG** -- advanced framework with parallel execution and data providers support
  - **Selenium** -- web browser automation
  - **REST Assured** -- REST API testing

  **JavaScript/TypeScript frameworks:**
  - **Playwright** -- modern E2E testing tool
  - **Cypress** -- E2E testing with built-in debugger
  - **Jest** -- JavaScript unit testing
  - **Vitest** -- fast framework for Vite projects

  **Python frameworks:**
  - **pytest** -- most popular Python testing framework
  - **Robot Framework** -- keyword-driven testing

  **Mobile testing:**
  - **Appium** -- cross-platform mobile app automation
  - **Espresso** -- native Android testing
  - **XCUITest** -- native iOS testing

  The choice of framework depends on the project's programming language, type of testing, and team needs.
