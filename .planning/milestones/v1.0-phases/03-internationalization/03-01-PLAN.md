Replace the entire content of `src/pages/index.astro` with a JS-based redirect that reads localStorage:

```astro
---
// Root URL redirects to stored locale preference or default (ua)
// Replaces meta refresh with JS redirect to support locale persistence
---
&lt;html&gt;
  &lt;head&gt;
    &lt;meta charset="utf-8" /&gt;
    &lt;title&gt;Redirecting...&lt;/title&gt;
    &lt;script is:inline define:vars={{ baseUrl: import.meta.env.BASE_URL.replace(/\/?$/, '/') }}&gt;
      (function() {
        var locale = localStorage.getItem('locale') || 'ua';
        if (locale !== 'ua' && locale !== 'en') locale = 'ua';
        window.location.replace(baseUrl + locale + '/');
      })();
    &lt;/script&gt;
    &lt;noscript&gt;
      &lt;meta http-equiv="refresh" content="0;url=ua/" /&gt;
    &lt;/noscript&gt;
  &lt;/head&gt;
  &lt;body&gt;&lt;/body&gt;
&lt;/html&gt;
```

Key points:
- Uses `window.location.replace()` instead of `window.location.href` to avoid polluting browser history
- Validates locale value (only 'ua' or 'en' accepted, defaults to 'ua')
- Uses `define:vars` to inject BASE_URL at build time (same pattern as existing codebase)
- Keeps noscript fallback with meta refresh pointing to /ua/ for users without JS
- Removes the old meta refresh in the head (replaced by JS + noscript fallback)
